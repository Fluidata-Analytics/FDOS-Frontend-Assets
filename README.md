# Frontend-Assets

## Purpose

The **Frontend-Assets** directory is a centralized, governed location for **static visual assets** used across FDOS frontend applications.

This folder is **strictly limited** to non-executable, non-code resources such as icons, images, and SVGs that are consumed by frontend UI layers.

---

## Asset Scope (Strict)

Only the following asset types are permitted:

- SVG icons 
- PNG / JPG / WebP images
- Brand logos and product marks
- UI-specific visual elements (backgrounds, loaders, placeholders)

❌ No JavaScript  
❌ No CSS  
❌ No fonts  
❌ No build artifacts  
❌ No environment or config files
